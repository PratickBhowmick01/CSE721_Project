# CSE721_Project
An interface to encrypt/decrypt 4 different cyphers and a Hill cypher cracker.

# Instructions on how to use use
1. Download the Zip file and extract at the desired folder.
2. Go to the folder's path and type 'cmd'. This will open the Command Prompt in that folder.
3. For the ciphers run 'python task1.py' and for the cracker tool run 'python task2.py'. Note that you need to have Python 3.10 or above installed in your OS.
